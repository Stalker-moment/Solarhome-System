# Solarhome-System
Control home with IoT

## Hardware Requirement :
- Arduino Mega
- NodeMcu Lolin V3 ESP8266
- INA219 Voltage Sensor
- 2x DC Voltage Sensor
- Solar Panel
- Solar charging module
- inverter
- PZEM AC voltage power sensor
- DHT22/DHT22
- 20x4 I2C LCD
- MQ7 & MQ6 sensor
- Soil sensor
- Motion (PIR) sensor
- Ultrasonic sensor (HC-SR04)
- Terminal Block
- Jumper cable
- 12V battery

  ## Intregrated :
- [Whatsapp Bot](https://github.com/Stalker-moment/Whatsapp-Bin)
